git基本使用

利用git上传文件

在GitHub上新建respository
复制git地址到本地想要上传的项目文件中，将项目文件复制到新建的文件中

git clone [url]

将文件添加进respository中：git add .
添加分支信息：git commit -m "自定义信息"
上传：git push -u origin master
